# BotMate - Platform for Building Chatbots

> BotMate is an open-source platform for building chatbots. It makes easier to build and deploy your bots with just few clicks. It supports Telegram, Discord, and Slack.

## Features

- **Easy to use**: BotMate is designed to be easy to use. You can build and deploy your bots with just few clicks.
- **Open-source**: BotMate is open-source and free to use. You can contribute to the project and make it better.
- **Multi-platform**: BotMate supports Telegram, Discord, and Slack.
- **Customizable**: You can customize your bots as you want. You can add new commands, features, and integrations.
- **Modern UI**: BotMate has a modern and clean user interface. It's easy to navigate and use.
- **Marketplace**: BotMate has a marketplace where you can find and install new plugins and integrations.

## Installation

You can install BotMate using Docker or manually. Check the [installation guide](https://docs.botmate.dev/installation) for more information.

## Tech Stack

BotMate is built using the following technologies:

- **Next.js** - Frontend (React Framework)
- **Temporal** - Workflow engine (for scheduling tasks)
- **Tailwind CSS** - CSS Framework
- **Next-UI** - UI Components
- **Prisma** - Database ORM
- **PostgreSQL** - Database
- **Docker** - Containerization
- **Nx** - Monorepo management
