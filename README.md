# botmate

> [!WARNING]
> The project is under constant development. There will be breaking changes.


Open source platform for building and hosting chatbots for Telegram, Discord and Slack, with aim of easy deployment.

## Roadmap

- [x] Telegram Support [in progress]
- [ ] Discord Support
- [ ] Slack Support
- [ ] Bot Analytics
- [ ] Plugin System - User can install/remove plugins using CLI
